Wecome to CloudBean!

This site is a single-page application (SPA) and was developed in Angular 13 with Bootstrap 5. You can find it in Github.

Feel free to clone it, Fork it, copy it and create your own repository.

This site is currently deployed in AWS and using the following AWS services:

  • AWS CloudFront – Content delivery network (CDN)
  • Route 53 – Scalable DNS and Domain Name Registration
  • AWS WAF - Web Application Firewall – Protects your web applications from common
  • AWS Shield – Provides AWS resources protection against Distributed Denial of Service (DDoS) attacks (Layer 3 and Layer 4)
  • AWS ACM - Certificate Manager – Allows the provision, management and deployment of public and private SSL/TLS certificates for use with AWS services
  • AWS Simple Storage Service (S3) – Scalable object storage
  • Amazon Cognito – Simple and Secure User Sign-Up, Sign-In, and Access Control

I'm not really putting tons of effort into the appearances, but wanted to give you the code from Gitgub and also to demonstrate how to use some AWS services.
